## [2026-02-23] v2.3 - Ontologize 看板 + 翻译优化

### ✨ 新功能

#### 新增 Ontologize 看板
- **频道**: https://www.youtube.com/@ontologize
- **视频数量**: 96 个
- **支持多看板系统**: Palantir、Ontologize、临时上传、极简上传
- **数据路径**: `frontend/public/data/ontologize/`

### ⚡ 性能优化

#### 翻译 API 并发优化
- **并发翻译**: 使用 5 线程并发翻译，速度提升约 **5倍**
- **分段策略**: 统一使用分段翻译（每段 1500 字符），避免长文本截断
- **优化前**: 22 段串行翻译，需要 1-2 分钟
- **优化后**: 22 段并发翻译，只需 10-20 秒

#### 修复中文翻译显示不全
- **问题**: 中文翻译被截断，只能显示部分内容
- **原因**: 单次翻译 max_tokens 限制导致长文本输出不完整
- **解决**: 实施分段翻译策略，确保所有内容完整翻译
- **代码位置**: `api.py:857-940`

### 🎨 用户体验改进

#### 前端加载提示优化
- **旋转加载动画**: 更明显的视觉反馈
- **友好提示文本**: "内容较多，翻译需要一些时间，请耐心等待..."
- **代码位置**: `frontend/src/components/RightSidebar.jsx:320-345`

### 🔧 技术改进

#### 后端并发翻译
- **实现**: 使用 `ThreadPoolExecutor` 实现并发翻译
- **限制**: 最多 5 个并发请求，避免 API 限流
- **代码位置**: `api.py:12, 857-940`

#### 多看板配置支持
- **配置文件**: `frontend/public/data/dashboards.json`
- **API 支持**: `/api/dashboards` 返回所有可用看板
- **动态路由**: 根据 `dashboard_id` 加载对应数据

### 📦 部署

#### 阿里云服务器部署
- **服务器**: root@59.110.21.174

- **服务重启**: API 服务已更新并重启
- **状态**: ✅ 生产环境正常运行

### 🐛 Bug 修复

#### 修复中文翻译截断问题
- **问题**: 长视频的中文翻译只能显示部分内容
- **影响**: 英文和双语模式正常，仅中文模式受影响

## [2026-02-14] v2.0 - Palantir 视频分析系统

### ✨ 核心功能

#### YouTube 频道分析
- **频道**: Palantir (@palantirtech)
- **功能**: 自动获取视频、分析、评分、分类

#### 视频评分系统
- **评分公式**: `Score = V_score × 0.4 + T_score × 0.3 + K_score × 0.3`
- **等级分类**: S (≥85)、A (≥70)、B (≥0)

#### 字幕提取
- **多级策略**: YouTube API → yt-dlp → BibiGPT → Whisper
- **本地化**: 支持中英文双语字幕

#### 智能报告
- **功能**: 基于视频内容生成 AI 分析报告
- **模板**: 支持自定义报告模板

### 🔧 技术栈

#### 后端
- **框架**: FastAPI (Python)
- **视频处理**: yt-dlp, youtube-transcript-api
- **AI**: OpenAI GPT, DeepSeek, BibiGPT
- **字幕**: Whisper (本地/API)

#### 前端
- **框架**: React + Vite
- **UI**: TailwindCSS
- **图表**: 可视化数据展示
